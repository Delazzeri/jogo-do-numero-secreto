Jogo do Número Secreto

Projeto desenvolvido com base nos estudos do curso de Lógica de Programação da Alura, com o objetivo de praticar conceitos fundamentais de JavaScript de forma interativa e divertida.
O jogo consiste em adivinhar um número secreto gerado automaticamente pelo sistema, utilizando tentativas e dicas até acertar o número correto.

Tecnologias utilizadas:
- HTML5
- CSS3
- JavaScript

Este projeto explora diversos conceitos importantes de programação, como:
- Estruturas condicionais
- Laços de repetição
- Manipulação do DOM
- Arrays
- Geração de números aleatórios
- Funções reutilizáveis
- Integração com a biblioteca ResponsiveVoice para leitura de textos
- Lógica de validação de entradas do usuário
